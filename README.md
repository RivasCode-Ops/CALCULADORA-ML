# Calculadora ML

Calculadora de venda do Mercado Livre para conferir lucro, taxas, frete e valor líquido real por pedido.

## Sobre o projeto

A CALCULADORA-ML nasceu para resolver um problema simples e comum de quem vende no Mercado Livre: vender bastante e ainda assim não saber exatamente quanto ganhou.

Na prática, o valor bruto da venda não representa o lucro real.
É preciso considerar taxa do Mercado Livre, frete, custo do produto, possíveis despesas extras e então descobrir quanto realmente sobra.

## Funcionalidades

- **Valor de Venda** - Preço do produto anunciado
- **Frete** - Custo do frete (pago pelo vendedor)
- **Taxa do Mercado Livre** - Percentual e taxa fixa configuráveis
- **Custo do Produto** - Quanto você pagou pelo produto
- **Outras Despesas** - Custos adicionais (embalagem, etc)
- **Lucro Líquido** - Valor real que sobra
- **Margem Percentual** - Porcentagem de lucro sobre a venda
- **Detalhamento** - Visão completa de todos os custos
- **Modo Simulação** - Simule preços antes de anunciar
- **Design Responsivo** - Funciona no celular e desktop
- **Dark Mode** - Suporte automático ao modo escuro

## Deploy

Este projeto está pronto para GitHub Pages:

1. Faça push para o repositório
2. Acesse Settings > Pages
3. Selecione Source: Deploy from a branch
4. Escolha a branch main e pasta / (root)
5. A URL será: `https://SEU-USUARIO.github.io/CALCULADORA-ML/`
